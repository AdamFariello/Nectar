# Nectar (On hiatus) 

## Introduction
--- 
### Project Description
Amazon assistant extension 
that lets the user save items to a list for:

- Price watching   
- Availability watch   
- Automatic purchases based on certain conditions   
- etc.   

Plus general features as well such as notifications
